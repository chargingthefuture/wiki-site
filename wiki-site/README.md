# Charging The Future Wiki Blog

[![Deploy Blog to GitHub Pages](https://github.com/chargingthefuture/wiki-site/actions/workflows/deploy-blog-gh-pages.yml/badge.svg?branch=main)](https://github.com/chargingthefuture/wiki-site/actions/workflows/deploy-blog-gh-pages.yml)

Static frontend for the Charging The Future wiki content.

Live site:

- https://chargingthefuture.github.io/wiki-site

Working directory:

- `/workspaces/chargingthefuture/wiki-site`

Common commands:

```bash
pnpm blog:validate
pnpm blog:sync
pnpm blog:preview
pnpm blog:build
pnpm blog:build:pages
```

GitHub Pages:

- Deploys from the `main` branch.
