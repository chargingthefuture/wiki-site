# Agent Skills Directory

This directory contains skills for use by agents in the CTF monorepo. Each skill is a self-contained folder with documentation, references, and integration notes.

- Place new skills in this directory following the structure:
  - skills/<skill-name>/SKILL.md
  - skills/<skill-name>/README.md
  - skills/<skill-name>/AGENTS.md
  - skills/<skill-name>/references/
